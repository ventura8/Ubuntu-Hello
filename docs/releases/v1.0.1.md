# 🚀 Ubuntu Hello v1.0.1 Release

Welcome to the **v1.0.1** release of **Ubuntu Hello**! This is a security-hardening and maintenance release focusing on resolving vulnerabilities, securing permissions, improving liveness detection robustness, and cleaning up legacy resources.

---

## ✨ Key Enhancements in v1.0.1

*   **🔒 Security Hardening**:
    *   **Input Validation**: Implemented strict regex-based username validation (`^[a-zA-Z0-9_.][a-zA-Z0-9_.-]*\$?$`) in both the C++ PAM module and Python CLI to prevent command injection and malformed argument attacks.
    *   **Directory Relocation**: Moved TPM context file generation (`primary.ctx`, `p_<pid>.ctx`, `s_<pid>.ctx`) from the globally-writable `/tmp` directory to a secured, root-only `/etc/ubuntu-hello/tpm-keys/` directory (`0700` permission controls), mitigating local symlink and directory harvesting attacks.
    *   **Secure Temporary Builds**: Updated the installer `install.sh` to use secure, randomized directory creation (`mktemp -d`) instead of predictable `/tmp` paths.
    *   **Subprocess Invocation Hardening**: Completely eliminated `shell=True` execution flags and `subprocess.getstatusoutput` in all administration helper scripts, switching to structured, list-based process execution.
*   **📁 File System Permissions**:
    *   Set strict root-only (`0700`) permissions for `/etc/ubuntu-hello/models/` and `/etc/ubuntu-hello/tpm-keys/` directories.
    *   Configured individual face model database `.dat` files with strict `0600` permissions (root-only access) upon creation.
*   **👁️ Nod Liveness Detection Fix**:
    *   Corrected a calculation error in the mathematical average formula for eye-distance tracking on initial frames, improving stability in nod-based liveness verification.
*   **🧹 Cleanup & Maintenance**:
    *   Purged legacy and redundant `debian` subdirectories (`ubuntu-hello/debian/` and `ubuntu-hello-gtk/debian/`) from the source tree to keep packaging definitions simple, clean, and secure.

## 📥 Installation

### Method 1: Quick Install (Recommended)
Open your terminal and run the universal installer:
```bash
curl -fsSL https://raw.githubusercontent.com/ventura8/ubuntu-hello/master/install.sh | sudo bash
```

### Method 2: PPA (Ubuntu / Linux Mint)
```bash
sudo add-apt-repository ppa:ventura8/ubuntu-hello
sudo apt update
sudo apt install ubuntu-hello
```

### Method 3: Debian / Manual Package
Download the attached `.deb` package for your architecture, then install it using `apt` or `gdebi`:
```bash
sudo apt install ./ubuntu-hello_1.0.1-1ppa1_amd64.deb
```
